# ChatGPT Example

## Recommended
- asdf
  - python 
  - poetry

Please refer to `.tool-versions` for the version

## Try

Setup
```
git clone git@github.com:daylight55/chatgpt-example.git
cd chatgpt-example
poetry install
```

Sets API key
```
cp .env.sample .env
# Set .env
vi .env
```

Run
```
poetry shell
.venv ‚ùØ python main.py
```
