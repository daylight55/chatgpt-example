

# CPU
## 正称
中央処理装置（ちゅうおんしょりせっけい、Central Processing Unit）の略。コンピュータの主要部品であり、プログラムを実行する役割を担っている。日本語では制御装置とも呼ばれる。CPUはコンピュータの処理能力を決定する要因の1つである。
